Módulo 1: Introducción a Excel

Conceptos básicos y fundamentos de las hojas de cálculo

Objetivo General:
Capacitar a los estudiantes en el manejo técnico y estratégico de Microsoft Excel, desarrollando habilidades para organizar, analizar y presentar datos con precisión.

Definición y conceptos básicos

Una hoja de cálculo es una cuadrícula gigante llena de filas y columnas. Cada "casilla" se llama celda y puede contener números, textos o fórmulas que hacen cálculos automáticamente.

Evolución histórica:

  • VisiCalc (1979): Primera hoja de cálculo digital
  • Lotus 1-2-3 (1983): Popular en los 80s
  • Excel (1985): Líder actual del mercado

Casos de uso

Excel se utiliza en diversos ámbitos:

  • Académico: Organizar notas, planificar proyectos, registrar datos de experimentos
  • Financiero: Contabilidad, presupuestos, análisis de inversiones
  • Administrativo: Listas de empleados, control de inventario, informes
  • Personal: Control de gastos, planificación de viajes, listas de compras

Ventajas de Excel

  • Automatización de cálculos: Las fórmulas se actualizan automáticamente
  • Organización clara: Datos estructurados en filas y columnas
  • Análisis de datos: Filtrado, ordenamiento y búsqueda rápida
  • Visualización: Creación de gráficos profesionales
  • Flexibilidad: Adaptable a múltiples necesidades

Módulo 2: Interfaz de Excel

Domina la interfaz y técnicas de navegación

Objetivo:
Explorar la estructura de Excel (celdas, filas y columnas) y aprender a navegar, seleccionar e ingresar información de manera eficiente.

Estructura básica

  • Columnas: Identificadas con letras (A, B, C... AA, AB, etc.)
  • Filas: Identificadas con números (1, 2, 3... hasta 1,048,576)
  • Celdas: Intersección de filas y columnas (ej. A1, B5)
  • Hojas: Cada pestaña en la parte inferior (Hoja1, Hoja2...)
  • Libro: Archivo completo que contiene las hojas (.xlsx)

Navegación eficiente

Atajos de teclado esenciales:

  • Flechas: Movimiento básico entre celdas
  • Ctrl + Flechas: Saltar al extremo de los datos
  • Ctrl + G: Ir a celda específica (ej. AZ5000)
  • Enter: Confirmar entrada y bajar a la siguiente celda
  • Tab: Confirmar entrada y mover a la derecha
  • F2: Editar celda activa

Ingreso y edición de datos

Tipos de datos:

  • Texto: Nombres, descripciones (alineado a izquierda)
  • Números: Cantidades, precios (alineado a derecha)
  • Fechas/Horas: Formatos especiales (25/06/2023, 10:30 AM)
  • Fórmulas: Siempre comienzan con =

Para editar: Doble clic en celda o presionar F2

Módulo 3: Fórmulas Básicas

Aprende a usar las funciones esenciales

Objetivo:
Aplicar funciones básicas para realizar cálculos automáticos que agilicen el trabajo y reduzcan errores.

Introducción a fórmulas

Todas las fórmulas en Excel comienzan con el signo =

Operadores básicos:

  • + (Suma): =A1+B1
  • - (Resta): =A1-B1
  • * (Multiplicación): =A1*B1
  • / (División): =A1/B1
  • ^ (Exponenciación): =A1^2

Ejemplo: =A1+B1*C1 (primero multiplica, luego suma)

Referencias de celdas

  • Relativas: Se ajustan al copiar (ej. A1)
  • Absolutas ($A$1): Fijas al copiar (para constantes)
  • Mixtas (A$1 o $A1): Fija solo fila o columna

Ejemplo práctico:

=B2*$C$1 (al copiar, B2 cambiará pero $C$1 permanecerá igual)

Funciones básicas

  • SUMA: =SUMA(A1:A10) - Suma un rango de celdas
  • PROMEDIO: =PROMEDIO(B2:B20) - Calcula la media
  • CONTAR: =CONTAR(C1:C100) - Cuenta celdas con números
  • MÁX: =MÁX(D2:D50) - Valor más alto
  • MÍN: =MÍN(E2:E50) - Valor más bajo

Autosuma: Botón Σ en la pestaña Inicio para sumar rápidamente

Módulo 4: Formato Condicional

Visualiza datos con formatos avanzados

Objetivo:
Utilizar herramientas para resaltar automáticamente ciertos valores según condiciones específicas.

Tipos de formato condicional

  • Reglas de resaltado: Colorear celdas según valores (ej. >100 en verde)
  • Barras de datos: Barras horizontales proporcionales al valor
  • Escalas de color: Degradados (ej. verde-amarillo-rojo)
  • Conjuntos de iconos: Semáforos, flechas, símbolos de estado

Ubicación: Pestaña Inicio > Formato condicional

Validación de datos

Restringe el tipo de datos que se pueden ingresar en una celda:

  • Números: Enteros/decimales dentro de un rango
  • Listas: Menú desplegable con opciones
  • Fechas: Dentro de un período específico
  • Textos: Con longitud determinada

Ubicación: Pestaña Datos > Validación de datos

Mensajes personalizados

Mensaje de entrada: Aparece al seleccionar la celda

Mensaje de error: Aparece si se ingresa dato inválido

Ejemplo: "Ingrese solo números entre 1 y 100"

Módulo 5: Gráficos

Crea gráficos profesionales

Objetivo:
Crear gráficos profesionales que permitan visualizar datos de forma clara y atractiva.

Tipos de gráficos

Principales tipos y cuándo usarlos:

  • Columnas/Barras: Comparar valores entre categorías
  • Líneas: Mostrar tendencias en el tiempo
  • Circular: Proporciones de un todo (máx. 5 categorías)
  • Dispersión (XY): Relación entre dos variables
  • Áreas: Mostrar cambios acumulativos

Creación de gráficos

Pasos básicos:

  1. Seleccionar los datos (incluyendo encabezados)
  2. Ir a pestaña Insertar > Gráficos
  3. Elegir el tipo de gráfico adecuado
  4. Personalizar diseño y formato

Elementos y personalización

Elementos clave:

  • Título del gráfico (claro y descriptivo)
  • Ejes con etiquetas y unidades
  • Leyenda (ubicación clara)
  • Etiquetas de datos (opcional)
  • Líneas de cuadrícula (para mejor lectura)

Diseño: Usar colores coherentes y evitar sobrecargar

Módulo 6: Tablas Dinámicas

Analiza grandes volúmenes de datos

Objetivo:
Aprender a resumir y reorganizar grandes volúmenes de datos mediante tablas dinámicas.

Preparación de datos

Requisitos para datos fuente:

  • Encabezados claros en cada columna
  • Sin filas o columnas completamente vacías
  • Datos consistentes (mismo formato en cada columna)
  • Preferible convertir a Tabla Excel (Ctrl+T)

Creación de tablas dinámicas

Pasos básicos:

  1. Seleccionar los datos (incluyendo encabezados)
  2. Ir a pestaña Insertar > Tabla dinámica
  3. Definir ubicación (nueva hoja recomendado)
  4. Arrastrar campos a las áreas:
    • FILAS: Lo que quieres agrupar (ej. Productos)
    • COLUMNAS: Subcategorías (ej. Meses)
    • VALORES: Datos a calcular (ej. Suma de Ventas)
    • FILTROS: Para filtrar datos globalmente

Funcionalidades avanzadas

  • Agrupación: Agrupar fechas por meses/trimestres
  • Cálculos: Mostrar valores como % del total
  • Ordenamiento: Ordenar por valores o etiquetas
  • Filtros: Segmentación de datos para filtrado visual
  • Actualización: Actualizar datos con nuevos registros

Módulo 7: Funciones Avanzadas

Búsquedas y operaciones condicionales

Objetivo:
Incorporar funciones complejas para búsquedas, operaciones condicionales y manejo de texto.

Función SI

Sintaxis básica:

=SI(condición, valor_si_verdadero, valor_si_falso)

Ejemplos:

  • =SI(A2>=60, "Aprobado", "Reprobado")
  • =SI(B2>100, B2*0.9, B2) (aplicar descuento si >100)

SI anidado:

=SI(A2>=90, "Excelente", SI(A2>=70, "Bueno", "Regular"))

Funciones de búsqueda

BUSCARV (Vertical):

=BUSCARV(valor_buscado, rango, columna, FALSO)

Ejemplo: =BUSCARV("Juan", A2:B10, 2, FALSO)

BUSCARH (Horizontal):

=BUSCARH(valor_buscado, rango, fila, FALSO)

INDICE + COINCIDIR (más flexible):

=INDICE(rango_resultados, COINCIDIR(valor_buscado, rango_busqueda, 0))

Funciones de texto

  • CONCATENAR: Unir textos (=CONCATENAR(A2," ",B2))
  • IZQUIERDA/DERECHA: Extraer desde extremos (=IZQUIERDA(A2,4))
  • EXTRAE: Extraer parte del texto (=EXTRAE(A2,3,5))
  • LARGO: Longitud del texto (=LARGO(A2))
  • ENCONTRAR: Posición de un texto (=ENCONTRAR("@",A2))

Módulo 8: Automatización

Macros para tareas repetitivas

Objetivo:
Explorar el uso de macros para grabar y ejecutar tareas repetitivas.

¿Qué es una macro?

Una macro es una serie de comandos e instrucciones que se agrupan como un solo comando para lograr una tarea automáticamente.

Ventajas:

  • Ahorra tiempo en tareas repetitivas
  • Reduce errores humanos
  • Automatiza procesos complejos

Grabación de macros

Pasos para grabar:

  1. Ir a pestaña Vista > Macros > Grabar macro
  2. Asignar nombre (sin espacios) y, opcionalmente, un atajo de teclado
  3. Realizar las acciones que deseas automatizar
  4. Detener grabación (Vista > Macros > Detener grabación)

Ejecución y gestión de macros

Ejecutar macro:

  • Vista > Macros > Ver macros > Seleccionar macro > Ejecutar
  • O usar el atajo de teclado asignado

Gestión:

  • Editar o eliminar macros desde el editor VBA (Alt + F11)
  • Guardar el archivo como .xlsm (Libro habilitado para macros)
