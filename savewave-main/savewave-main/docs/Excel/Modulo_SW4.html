What is Conditional Formatting?

Conditional formatting is a powerful tool that allows you to apply specific formats (colors, icons, data bars) to cells or ranges based on defined conditions or criteria.

[Image: Example of conditional formatting applied to a spreadsheet]

This feature is especially useful for:

  • Visualizing trends and patterns in your data
  • Quickly identifying outliers or critical values
  • Automatically highlighting important information
  • Improving data readability and analysis

Types of Conditional Formatting

Color Scales

Apply a color gradation based on the value of the cells. Useful for visualizing distributions and variations in data.

[Image: Example of a color scale applied to numerical data]

Data Bars

Insert horizontal bars within cells, where the length represents the value in relation to other data.

Icon Sets

Display icons or visual symbols according to predefined ranges (traffic lights, flags, stars, etc.).

[Image: Example of icon sets applied to a table]

Formula-Based Rules

Allow you to create custom conditions using formulas for complete control over the applied format.

Pro tip: Combine multiple conditional formatting rules to create visually striking and highly functional dashboards.

Data Validation

Data validation restricts the type of information that can be entered into a cell, ensuring data integrity and consistency.

[Image: Data validation dialog box showing different options]

Types of Validation

Drop-down lists

These allow you to select values from a predefined list, preventing typing errors.

Numeric Validation

Restricts entries to numbers within a specific range or that meet certain conditions.

Text validation

They limit the length of the text or require specific patterns.

Date validation

Ensure that entries are dates within a specified period.

Example of custom validation:
=Y(ISNUMBER(B2), B2>=0, B2<=100)
Only allow numbers between 0 and 100

Best Practices

Plan before applying

Clearly define what you want to highlight and why. Excessive conditional formatting can make reading difficult.

Use meaningful colors

Assign colors that make sense in your context (red for alerts, green for positive values, etc.).

[Image: Appropriate color palette for conditional formatting]

Document your rules

Keep a record of the rules you apply, especially when working with complex formulas.

Combine validation with formatting

Use conditional formatting to highlight cells that do not meet validations, even after they have been entered.

Remember: Data validation prevents errors, while conditional formatting helps you visualize them. Use both tools together.
