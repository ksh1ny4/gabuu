CONCATENATE FUNCTION

The CONCATENATE function joins two or more text strings into a single string. It is very useful for combining first and last names, addresses, or any information that is separated into different cells.

=CONCATENATE(text1, [text2], ...)

IF FUNCTION

The IF function performs a logical test and returns one value if the test is true and another value if it is false. It is ideal for creating conditions in your spreadsheets.

=IF(logical_test, value_if_true, [value_if_false])

VLOOKUP FUNCTION

VLOOKUP searches for a value in the first column of a range and returns a value in the same row from the column you specify. It is perfect for searching for information in large tables.

=VLOOKUP(lookup_value, table_array, column_index_number, [approximate_match])
Visual example: Product table with VLOOKUP finding the price of a specific product

IF.SET FUNCTION

The IF.SET function evaluates multiple conditions and returns the value corresponding to the first true condition. It is very useful when you have more than two possible conditions.

=IF.MULTIPLE(test1, value1, [test2, value2], ...)

FIND FUNCTION

The FIND function returns the position of one text within another text. It is case sensitive.

=FIND(search_text, text_to_search, [start_num])
