Introduction to formulas in Excel

Image space: Excel formula bar with a SUM function

Recommendation: Screenshot showing how to write a formula

What is a formula?

  • An expression that performs calculations on the data in your spreadsheet.
  • It always starts with the = (equal) sign.
  • It can contain numbers, cell references, operators, and functions.
  • Example: =A1+B1 Add the values of cells A1 and B1.

Basic arithmetic operators:

Operator Function Example Result
+ Addition =A1+B1 Adds the values of A1 and B1
- Subtract =A1-B1 Subtract B1 from A1
* Multiplication =A1*B1 Multiply A1 by B1
/ Division =A1/B1 Divide A1 by B1

Practical example:

To calculate total daily sales where B2, C2, and D2 are sales per shift:

=B2+C2+D2

Cell references

Space for image: Example of relative and absolute references

Recommendation: Screenshot showing how references change when copying formulas

Relative references:

  • They adjust automatically when copying the formula to other cells.
  • Example: If you copy =A1+B1 from row 1 to row 2, it will become =A2+B2.

Absolute references (with $):

  • Fix the column or row when copying.
  • Example: =$A$1+B1 will always use cell A1, even if the formula is copied to another row.

When to use them?

Absolute

For constants such as exchange rates, fixed factors, etc.

=$B$1*A2

Where $B$1 contains a fixed rate that applies to different values in column A.

Relative

For repetitive calculations that must be adapted to each row/column.

=A2*B2

When copied to row 3, it becomes =A3*B3, etc.

Basic Excel functions

Space for image: Function menu in Excel or example of use

Recommendation: Screenshot showing the function wizard or auto sum
Function Description Syntax Example
SUM Adds a range of cells =SUM(range) =SUM(A1:A10)
AVERAGE Calculates the average of values =AVERAGE(range) =AVERAGE(B2:B20)
COUNT Counts cells with numbers =COUNT(range) =COUNT(C1:C100)
MAX Returns the highest value =MAX(range) =MAX(D2:D50)
MIN Returns the lowest value =MIN(range) =MIN(E2:E50)

Real-world application:

Grades

=AVERAGE(F2:F30)

Calculates the average grade for a group of students.

Sales

=MAX(G2:G100)

Identifies the day with the highest sales in a quarter.

Using AutoSum

Space for image: AutoSum button in Excel

Recommendation: Screenshot showing the AutoSum drop-down menu

How does it work?

  1. Select the cell where you want the result (usually below a column of numbers or at the end of a row).
  2. Click the AutoSum (Σ) button on the Home tab.
  3. Excel will suggest a range. Check that it is correct.
  4. Press Enter to confirm.

Advantages:

  • Fast and efficient for adding entire columns/rows.
  • Also works with other functions such as AVERAGE, COUNT, MAX, MIN (by clicking the arrow next to Σ).
  • Automatically detects the range to calculate based on adjacent data.

Formula bar and function wizard

Formula bar

Displays and edits the contents of the active cell.

It is especially useful for checking and editing long and complex formulas.

Function assistant (fx)

Helps you insert functions correctly.

  1. Click fx next to the formula bar.
  2. Choose a function (e.g., COUNTIF).
  3. Complete the arguments guided by Excel.

💡 Pro tip

Use descriptive names for your data ranges. Instead of using =SUM(A1:A10), you can name the range “Sales_Quarter1” and use =SUM(Sales_Quarter1), which makes your formulas more readable and easier to maintain.
